Smart Eraser

Abstract

The main purpose of the smart eraser is simulating the eraser effect

in a given video. Meanwhile, this project is very primitive by using target tracking

based on bouding box and dynamic programming to simulate the erased part.

Target Tracking

To begin with, the location and proper size of bounding box

including the eraser should be given in the first frame of video.

And it is easy to see a sentence "Can you erase me?",

written in red pen, which is the erased goal of this project.

With the tracking criterion, the bounding box would follow the movement

of eraser and provide a coordinate to the next step of eraser effect.

Dynamic Programming for Eraser Simulation

The dynamic programming is simulating a coherent erased area to get nearly real eraser effect.

First it would obtain enough samples from clean background such as the drawn area in left picture.

Then the DP technique would calculate the most suitable and coherent boundary.

Summary

The results are natural as expected with the good performance of DP.

There exist some incoherent boundary and the work can be further improved

with trials of different methods in target tracking and eraser simulation.
